# 🎄 Merry Christmas - 3D 粒子圣诞树

一个浪漫的3D粒子圣诞树网页应用，使用 Three.js 创建，作为特别的圣诞礼物。

## ✨ 功能特性

- 🎨 **3D 粒子圣诞树**：由 4000+ 发光粒子组成的圣诞树
- 🌟 **树顶星星**：金色闪烁的星星
- 🎁 **彩色装饰球**：红色、金色、蓝色、粉色装饰
- ❄️ **飘落雪花**：800+ 雪花粒子持续飘落
- ✨ **背景星星**：闪烁的星空背景
- 🎮 **交互控制**：
  - 旋转速度控制（0-5）
  - 树的高度调节（50-150）
  - 粒子大小调节（0.5-3）
- 💝 **浪漫水印**：专属 "To:  wyw ❤️"

## 📦 本地运行

### 方法 1：直接打开（推荐）
直接双击 `index.html` 文件即可在浏览器中打开。

### 方法 2：使用本地服务器
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx http-server
```

然后访问 `http://localhost:8000`

## 🌐 部署到 GitHub Pages

1. 将所有文件推送到 GitHub 仓库
2. 进入仓库设置：`Settings` → `Pages`
3. 在 `Source` 下选择 `main` 分支
4. 点击 `Save`
5. 等待 1-2 分钟，你的网站将发布在：
   `https://<username>.github.io/<repository-name>/`

## 🎨 设计细节

### 颜色方案
- 圣诞树：深绿 → 中绿 → 亮绿渐变
- 装饰球：红、金、蓝、粉
- 背景：黑灰渐变 (#1a1a1a → #0a0a0a)
- 重点色：金色 (#ffd700)

### 粒子数量
- 圣诞树：4000 个粒子
- 雪花：800 个粒子
- 背景星星：200 个粒子
